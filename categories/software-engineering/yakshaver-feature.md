# Rule - Using YakShaver Desktop App for Efficient AI-assisted Development

## Overview
Discover how to effectively utilize the YakShaver desktop app's new feature for AI-assisted development. This includes the ability to view current application versions and easily switch between PR pre-releases, enabling fast testing before production deployment.

## Details
- **Feature:** YakShaver desktop app now includes a "Releases" section in its settings.
- **Functionality:** Users can view the current version of the app and switch between PR pre-releases.
- **Benefits:** This feature makes AI-assisted development faster and more efficient, allowing for rapid feature testing and more frequent production releases.

## Why It Matters
With AI-assisted development, you have the ability to create multiple PRs quickly and test features efficiently. The new release management makes development workflows not just quicker, but also smarter, facilitating continuous delivery.
