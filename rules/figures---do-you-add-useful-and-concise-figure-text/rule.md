<span class='intro'> <p>When you add an image to a website or application, it is so useful to add a figure underneath it to describe your image.&#160;<br>(This is the same as how Microsoft Word does it when you add a caption.)<br></p><p>It's the best way of ensuring you catch users' attention to the content of your page. When you're scanning a newspaper for interesting articles, you'll check out the pictures, read the accompanying description, and if it sounds interesting you'll go back and read the article.</p>
                     </span>

<p>Users read websites in a similar fashion. Catch their attention with an image, and then keep it with a useful description. Don't just describe what the image is; say what it's used for in the context of the document.
