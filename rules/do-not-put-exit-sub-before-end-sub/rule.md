This is often a bad practice if you already are ending Sub you don't need another line.

<!--endintro-->



```
Sub MySub
…
End Sub
Exit sub
```






::: bad
Figure: Bad example  
:::



```
Sub MySub
…
End sub
```






::: good
Figure: Good example

:::
