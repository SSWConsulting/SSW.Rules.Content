Document libraries in SharePoint 2016 are much better. The keyboard shortcuts are:

<!--endintro-->

* Alt +  **N**  -  **N** ew
* Alt +  **E**  -  **E** dit
* Alt +  **U**  -  **U** pload
* Alt +  **M**  -  **M** anage
* Alt +  **S**  -  **S** hare
* Alt +  **Y**  - Synchronization
