<span class='intro'> <p>Often there are two email threads that have separate tasks in them, but that both relate to a very similar topic. In this case, it can be useful to give them the same subject, with a number afterwards to differentiate them.</p> </span>

<p>​​Example&#58;</p><p class="greyBox">SSW Rules - Add 'please update this rule' button <span class="ssw15-rteStyle-Highlight">#1</span></p><p class="greyBox">SSW Rules - Add 'please update this rule' button <span class="ssw15-rteStyle-Highlight">#2</span></p><p class="p4"><span class="s1">Read <span class="s2"><a href="/Pages/ImportanceOfAGoodSubject.aspx">Do you realize the importance of a good email Subject?</a></span>&#160;for more on good email subjects.​</span></p>
