Before upgrading from CRM 2011 to CRM 2013, need to review migration doesn't contain any legacy code.

<!--endintro-->

You can use [legacy check tool](http://go.microsoft.com/fwlink/p/?LinkID=309565) to verify it before migration. [More details](https://download.microsoft.com/download/4/F/D/4FD108C5-4C52-4F82-AB9F-9FA3337A0369/How%20to%20Prepare%20for%20the%20CRM%20Online%20Fall%20%2713%20Service%20Update.pdf).
