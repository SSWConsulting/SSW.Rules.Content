When quoting prices, it's common practice to state the monthly cost. However, it's crucial to provide a full picture by also including the annual cost.
            
<!--endintro-->

::: greybox
AUD $150 + GST/month
:::
::: bad
Figure: Bad example - Only include monthly cost
:::

::: greybox
AUD $150 + GST/month (AUD $1,800 + GST/year)
:::
::: good
Figure: Good example - Include both monthly and yearly cost
:::
