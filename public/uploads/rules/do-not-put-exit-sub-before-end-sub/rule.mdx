This is often a bad practice if you already are ending Sub you don't need another line.

<endOfIntro />

```vbnet
Sub MySub
...
End Sub
Exit sub
```
**❌ Figure: Bad example**  

```vbnet
Sub MySub
...
End sub
```
**✅ Figure: Good example**
