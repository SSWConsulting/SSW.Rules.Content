Events should end in "ing" or "ed".

<endIntro />

```csharp
public event Action
< connectioninformation > ConnectionProblem;
```
**❌ Bad example**  

```csharp
public event Action
< connectioninformation > ConnectionProblemDetected;
```
**✅ Good example**
