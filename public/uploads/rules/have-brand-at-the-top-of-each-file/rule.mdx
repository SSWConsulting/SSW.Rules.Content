The brand should contain the copyright declaration.

```vs
///<summary>
///'----------------------------------------------
/// Copyright 2017 SSW
/// www.ssw.com.au All Rights Reserved.
///'----------------------------------------------
/// Comment: User class to handle user preference and login information
/// Authors:   DDK,PH
/// Reviewers: AC,RD
///</summary>
///'----------------------------------------------
```
