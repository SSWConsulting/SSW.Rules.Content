ChatGPT, custom GPTs, and the OpenAI API all use OpenAI models, but they are best suited to different scenarios.

<endIntro />

## ChatGPT

Use ChatGPT when a person wants to interact with AI directly.

Good examples:

* Drafting or rewriting an email
* Explaining a concept
* Brainstorming ideas
* Summarizing a document
* Asking follow-up questions
* Getting help with a once-off task

✅ Best for human-in-the-loop work.

## Custom GPT

Use a custom GPT when you want a repeatable assistant inside ChatGPT with specific instructions, knowledge, tools, or actions.

Good examples:

* An internal SSW Rules assistant
* A sales proposal helper
* A support triage assistant
* A reusable assistant trained on uploaded reference files
* A team-specific helper with a consistent tone and process

✅ Best for repeatable team workflows where people still interact through ChatGPT.

## OpenAI API / GPT models

Use the OpenAI API when you need automation, product integration, or system-to-system workflows.

Good examples:

* Summarizing pages automatically
* Turning a URL into a draft SSW Rule
* Rewriting email content for different audiences
* Building AI features into an app
* Running AI as part of a backend workflow
* Connecting AI to your own database, services, or business logic

✅ Best for production systems, automation, and programmatic integration.
