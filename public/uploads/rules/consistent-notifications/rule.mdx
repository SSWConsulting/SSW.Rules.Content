Users miss important updates when notifications are hidden, inconsistent, or unclear. A standard notification pattern ensures users instantly recognize and access alerts.

<endIntro />

## Place notification buttons next to user avatars

Position the notification button beside the user avatar. For most websites that use a top navbar, this is in the top-right corner. For websites that use a sidebar, place it in the bottom-left corner.

This follows common UI conventions and groups account-related actions logically.

## Use a bell icon for notifications

Avoid envelopes (email), abstract symbols, or text-only labels. The bell is universally recognized and reduces confusion.

## Show a notification count badge

Display a small badge on the bell icon to show unread notifications.

Best practices:

* Use a red (or brand-colored) circular badge  
* Position it at the top-right of the bell  
* Show a number (e.g. ‘3’, or ‘99+’ for large counts)
