You should avoid any extra style tags in your heading text, because it is unnecessary and generates inconsistency.
![Bad example – bold tags being used within header tags.](/uploads/rules/do-you-know-not-to-use-bold-tags-inside-headings/heading-with-bold-tags.png)
![Good example – all styling is being done through CSS.](/uploads/rules/do-you-know-not-to-use-bold-tags-inside-headings/heading-styled-with-css.png)
  </>}

**Tip:** You can do all the styling via CSS.
