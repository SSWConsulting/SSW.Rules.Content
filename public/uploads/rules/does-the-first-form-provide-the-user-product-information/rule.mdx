The main menu of the application should have a **Help | About...** menu item. When clicked, it should show the first few paragraphs of the application's website often contain virtually identical information (i.e., describing the product).

<endIntro />

## For wizard applications

The first page of the utility should introduce the user to the application.

The first two paragraphs of the screen can be copied from portions of the website.

A "More" hyperlink can also be added at the bottom of the screen, which will direct the user to the website where they can read further information about the application.
