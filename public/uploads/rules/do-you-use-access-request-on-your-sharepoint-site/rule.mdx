<introEmbed
  body={<>
Instead of displaying a direct " **Access Denied** " warning info, you can allow end users to send an " **Access Request** ".

![Figure: Joanna is requesting access to SharePoint site](/uploads/rules/do-you-use-access-request-on-your-sharepoint-site/PermissionRequest.jpg)
  </>}
/>
The "request manager" will receive an email:


<imageEmbed
  alt="Image"
  size="large"
  showBorder={false}
  figureEmbed={ {
    preset: "default",
    figure: "Request Notification Email Sample",
    shouldDisplay: true
  } }
  src="/uploads/rules/do-you-use-access-request-on-your-sharepoint-site/637cf8_RequestNotificationEmail.png"
/>


The link in the email will navigate administrator to the **Pending Requests** list:


<imageEmbed
  alt="Image"
  size="large"
  showBorder={false}
  figureEmbed={ {
    preset: "default",
    figure: "Pending Requests List",
    shouldDisplay: true
  } }
  src="/uploads/rules/do-you-use-access-request-on-your-sharepoint-site/LinkToPendingRequestsList.png"
/>


After reading the request infomation, the administrator can "Approve" or "Decline" the request, or he can start a conversation with the request user on the **Pending Requests** list directly to inquire more information:


<imageEmbed
  alt="Image"
  size="large"
  showBorder={false}
  figureEmbed={ {
    preset: "default",
    figure: "possible actions for requests (Approve, Decline or start a conversation with the request user)",
    shouldDisplay: true
  } }
  src="/uploads/rules/do-you-use-access-request-on-your-sharepoint-site/StartAConversatioinOnPendingList.png"
/>


To setup permission request for a SharePoint site collection, go to " **Site Settings (Gear Wheel icon)** | **Site Permissions** ":


<imageEmbed
  alt="Image"
  size="large"
  showBorder={false}
  figureEmbed={ {
    preset: "default",
    figure: "Open \"Access Request\" setting",
    shouldDisplay: true
  } }
  src="/uploads/rules/do-you-use-access-request-on-your-sharepoint-site/SetupPermissionRequest.png"
/>


**Limition:** This "Access Request" only works for authenticated users to inquire more access permission, that means if your site allows "anonymous access", then an anonymous user cannot send "access request" as he doesn't have an identify to be assigned more access permission.
